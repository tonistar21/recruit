# Залежності

| Бібліотека | Призначення | Причина |
|---|---|---|
| React Router | маршрути/layouts | стандартний declarative routing |
| TanStack Query | API cache/states | контроль loading/error/refetch |
| React Hook Form + Zod | форми | типізована клієнтська валідація |
| Recharts | Dashboard | доступні React-графіки |
| Lucide React | іконки | легкий узгоджений набір |
| Sonner | повідомлення | доступні toast states |
| FastAPI + Pydantic | REST/валідація | OpenAPI та строгі схеми |
| SQLAlchemy + Alembic | ORM/міграції | транзакції та PostgreSQL migrations |
| PyJWT + Argon2 | сесія/паролі | перевірені криптографічні примітиви |
